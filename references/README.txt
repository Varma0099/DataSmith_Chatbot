NEPHROLOGY REFERENCE MATERIALS
================================

Place your nephrology PDF files in this directory.

Recommended Sources:
- National Kidney Foundation guidelines
- ISN (International Society of Nephrology) materials
- Handbook of Dialysis sample chapters
- Clinical nephrology textbooks
- KDIGO guidelines

Supported Format: PDF

The system will automatically:
1. Load all PDF files from this directory
2. Extract text content
3. Create embeddings
4. Store in ChromaDB vector database

If no PDFs are provided:
- The system will use the built-in sample nephrology knowledge base
- This is sufficient for demonstration purposes
- Sample content covers all major nephrology topics

File Naming Suggestions:
- nephrology_guidelines.pdf
- kidney_disease_management.pdf
- dialysis_handbook.pdf
- ckd_clinical_guide.pdf

After adding PDFs:
- Delete the chroma_db/ folder
- Restart the application
- The system will rebuild the vector database with your new content

Note: Ensure you have proper rights to use any medical reference materials.
