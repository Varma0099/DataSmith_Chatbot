STEP 1: Install Dependencies
----------------------------
# Open PowerShell/Command Prompt in this directory

# Create virtual environment
python -m venv venv

# Activate it (IMPORTANT!)
venv\Scripts\activate

# Install packages
pip install -r requirements.txt


STEP 2: Configure API Keys
--------------------------
# Copy the example environment file
copy env.example .env

# Edit .env file and add your Google Gemini API key:
GOOGLE_API_KEY=your-actual-key-here

Get your FREE Gemini API key at: https://makersuite.google.com/app/apikey
(No credit card required!)


STEP 3: Verify Setup (Optional)
-------------------------------
python setup.py


STEP 4: Launch Application
--------------------------
# Use the convenient launch script
run.bat

# Or run directly
streamlit run app_streamlit.py

The app will open in your browser at http://localhost:8501


STEP 5: Test It!
----------------

3. Ask: "What should I eat?"
4. Get medical advice!

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "ModuleNotFoundError"
Solution: Make sure venv is activated and run: pip install -r requirements.txt

Problem: "Google API Error"
Solution: Check your API key in .env file (get FREE key: https://makersuite.google.com/app/apikey)

Problem: "Port already in use"
Solution: Run: streamlit run app_streamlit.py --server.port 8502

Problem: ChromaDB errors
Solution: Delete chroma_db folder and restart
